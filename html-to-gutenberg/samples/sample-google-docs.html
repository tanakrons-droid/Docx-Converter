บทความตัวอย่าง: วิธีใช้งาน HTML to Gutenberg Converter

บทความนี้จะแนะนำวิธีการใช้งานเครื่องมือแปลง HTML เป็น Gutenberg blocks

ขั้นตอนที่ 1: การติดตั้ง

ก่อนอื่นให้ติดตั้ง dependencies ด้วยคำสั่ง:

npm install

ขั้นตอนที่ 2: การใช้งาน

หลังจากติดตั้งเสร็จแล้ว สามารถใช้งานได้ดังนี้:

  • เตรียมไฟล์ HTML ที่ต้องการแปลง
  • รันคำสั่ง npm run convert
  • ตรวจสอบผลลัพธ์ในไฟล์ output
